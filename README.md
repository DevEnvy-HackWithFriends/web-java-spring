Hack With Friends - DevEnvy
===========================

Introduction
------------

DevEnvy is an implementation of Eclipse Che.

Installation
------------

# Install docker
# Check out this git repository

Start DevEnvy
-------------
   
Run the following command:

    ./cheUp.sh

Check Status
------------

Run the following command:

    ./cheStatus.sh

Stop DevEnvy
------------

Run the following command:

    ./cheDown.sh
